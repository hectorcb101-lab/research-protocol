# Research Protocol - Video Demo Plan

## Video Concept: "Research with Receipts"

**Duration:** 60-90 seconds
**Style:** Technical demo with animations
**Tool:** Remotion (React-based video)

## Scenes

### Scene 1: The Problem (0-15s)
- Text: "AI agents can research anything..."
- Animation: Agent icon researching
- Text: "But how do you trust the output?"
- Show: Cherry-picked sources, modified data

### Scene 2: The Solution (15-35s)
- Text: "Research Protocol: Commit-Reveal Research"
- Animation: 4-step flow
  1. Commit methodology BEFORE research
  2. Fetch and hash sources
  3. Publish with audit trail
  4. Anyone can verify

### Scene 3: Live Demo (35-55s)
- Show: Real terminal output
- Hash commits appearing
- Sources being fetched
- Verification checkmarks

### Scene 4: Results (55-70s)
- Text: "3 Research Reports Published"
- Show: Report titles scrolling
- All with committed methodology
- Full verification trails

### Scene 5: CTA (70-90s)
- Text: "Research with Receipts"
- GitHub link
- Colosseum project link

## Technical Requirements
- Remotion + React
- Terminal animation component
- Hash visualization
- Smooth transitions
- Clean typography

## Assets Needed
- Terminal screenshots/recordings
- Report hashes
- Flow diagram
- Logo/branding
